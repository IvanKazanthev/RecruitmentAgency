# Recruitment Agency Database

## Описание

Приложение для управления базой данных соискателей кадрового агентства. Система демонстрирует практические навыки работы с реляционными БД, сортировкой данных и обработкой информации.

## Функциональность

Приложение работает с информацией о соискателях:
- ФИО, пол, дата рождения
- Специальность, стаж работы
- Знание иностранных языков
- Ожидаемый оклад

### Доступные отчеты:

1. **Полный список соискателей** (сортировка по специальности и фамилии)
2. **Поиск по специальности** (сортировка по стажу, полу, фамилии)
3. **Фильтр по диапазону оклада** (сортировка по оплате и фамилии)

## Технические детали

- **Язык:** Python
- **Алгоритм сортировки:** Shaker Sort (сортировка коктейлем)
- **Объем данных:** 25+ записей соискателей
- **Парадигма:** Объектно-ориентированное программирование
